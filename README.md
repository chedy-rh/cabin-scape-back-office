# CabinScape Back Office

This the back office of the CabinScape web app which is a luxury cabins booking application

## Used Technologies

* React 19.2
* React Router V7
* Styled components V6
* Tanstack React Query V5
* Supabase (back-end) 

## Running Steps

1 - Clone the project  
2 - Run pnpm install  
3 - Use your supabase credentials (Optional)  
4 - Run pnpm dev
